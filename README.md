# YouCanCallMeMomo.github.io 🚀

Selamat datang di repositori utama portofolio dan proyek web statis! 
Repositori ini berfungsi sebagai wadah penyimpanan *(monorepo)* untuk beberapa aplikasi web interaktif dan portal edukasi yang dikembangkan menggunakan berbagai teknologi *frontend* modern. 

Aplikasi di bawah ini diservis langsung secara otomatis dengan **GitHub Pages**.

## 📁 Daftar Proyek

Berikut adalah proyek-proyek yang tersedia di dalam repositori ini:

### 1. 🛡️ [CyberBro (SiagaSiber Youth System)](./CyberBro/)
**Tautan Langsung:** [Akses Web CyberBro!](https://youcancallmemomo.github.io/CyberBro/build/)

SiagaSiber Youth System adalah sebuah platform edukasi **Keamanan Siber (*Cyber Security*) interaktif** yang dirancang khusus untuk pelajar SMP dan SMA di Indonesia. Dengan pendekatan gamifikasi modern, platform ini menjadikan belajar *cyber defense* sangat asyik dan seru.

**Fitur Utama:**
- **Mini-Games:** *The Leaked Vault, The Chat Trap, Defacement Scanner, Forensic Investigator, dan SQLi Scanner.*
- **Kuis Acak:** Ratusan simulasi soal Cybersecurity.
- **CyberBro AI:** Chatbot kecerdasan buatan untuk menjawab pertanyaan keamanan siber secara non-stop.
- *Tech Stack:* React (TypeScript), Vite, TailwindCSS, Framer Motion.

---

### 2. 📚 [StudyHub (Interactive Study Portal)](./studyhub/)
**Tautan Langsung:** [Akses Web StudyHub!](https://youcancallmemomo.github.io/studyhub/)

StudyHub adalah sebuah portal belajar komprehensif yang dirancang untuk persiapan ujian tengah semester mahasiswa. Portal ini merangkum materi perkuliahan dalam format yang modern, elegan, dan dilengkapi dengan bank kuis interaktif yang memberikan *feedback* instan.

**Fitur Utama:**
- **Mode Presentasi (*Slide*):** Materi markdown diuraikan secara otomatis menjadi slide pembelajaran ala "PowerPoint" interaktif.
- **Bank Kuis Interaktif:** Uji pemahaman dengan kuis instan pada mata kuliah:
  - *Software Engineering* (SOFENG)
  - *Software Security* (SOFTSEC)
  - *Server & Network Administration* (SNA)
  - *Biology*
- *Tech Stack:* HTML5 Murni, Vanilla JS, CSS Animations.

## 🚀 Cara Mengakses Proyek Secara Lokal

Jika kamu ingin menjalankan atau mengubah proyek ini secara lokal di komputermu:

1. Lakukan klon repositori ini:
   ```bash
   git clone https://github.com/YouCanCallMeMomo/YouCanCallMeMomo.github.io.git
   ```
2. Untuk proyek **StudyHub** (HTML statis):
   Kamu cukup membuka file `studyhub/index.html` menggunakan *web browser* atau ekstensi Live Server di VSCode.
3. Untuk proyek **CyberBro** (React + Vite):
   - Masuk ke direktori: `cd CyberBro`
   - Instal dependensi: `npm install`
   - Jalankan server lokal: `npm run dev`
